# Score Tracker (Football)

A sleek, responsive web application designed for football enthusiasts and referees to track match scores easily. Features an intuitive user interface modeled after a football pitch to manage live match setups seamlessly.

## Features
* **Quick Match Setup:** Easily input names for Team 1 and Team 2 to initialize a fixture.
* **Intuitive Interface:** Immersive football pitch visual theme for an engaging user experience.
* **Live Score Tracking:** Streamlined flow to start matches and manage real-time updates.
